

### Purpose
A Next JS Application for RAG Web site, where user can upload the document and ask questions related to the document.

### Features
- User can upload the document and ask questions related to the document.
- User can ask questions related to the document.

### Backend
- Have a Python server where we have api endpoints like
  1. upload the document - /api/upload
  2. ask questions related to the document - /api/custom-chat

### Frontend
- Next JS with Tailwind CSS.
